# brokenAccessControlExample
This is a Project for the Modul 183 where we had to explain the Broken Access Control Category from the OWASP Top 10 list

# Get started
to use this project execute the following commands:

go to the docker folder
```
cd docker
```

and execute docker-compose file
```
docker-compose up
```

after that a docker container with apache web server gets spinned up. You should be able to see the application on localhost:8000
